# Privacy Policy for PlanYT

**Last Updated: February 2026**

## 1. Overview

PlanYT is a Chrome extension designed to help users plan and manage YouTube playlists by automatically splitting videos into manageable daily goals and tracking progress locally within the browser.

Your privacy is important. PlanYT is built with a privacy-first approach and operates entirely within your browser.

---

## 2. Information We Collect

PlanYT does **not** collect, store, or transmit any personally identifiable information.

The extension may access and process the following data locally within your browser:

- YouTube playlist metadata (e.g., video titles, durations, and playlist structure)
- User-created planning configurations
- Progress tracking information related to playlist completion

This data is processed only to provide the core functionality of the extension.

---

## 3. How Data Is Used

All data accessed by PlanYT is used strictly to:

- Generate structured daily viewing plans
- Track watch progress
- Restore active plans
- Improve user experience within the extension

Data is processed locally in your browser and is not transmitted to any external servers.

---

## 4. Data Storage

PlanYT uses the Chrome `storage` API to store:

- Playlist planning metadata
- Progress state
- Active plan restoration information

All stored data remains on the user's device and is never sent to any third-party servers.

---

## 5. Data Sharing

PlanYT does **not**:

- Sell user data
- Share user data with third parties
- Transfer data outside the browser
- Use data for advertising or tracking
- Use data for profiling or analytics

---

## 6. Remote Code

PlanYT does not execute remote code.  
All executable scripts are bundled within the extension package distributed through the Chrome Web Store.

---

## 7. Third-Party Services

PlanYT does not use:

- Analytics tools
- Advertising networks
- External APIs that collect user data
- Tracking scripts

---

## 8. Permissions Explanation

PlanYT requests access to `https://www.youtube.com/*` solely to read playlist metadata necessary for generating viewing plans.

No other websites are accessed.

---

## 9. Children's Privacy

PlanYT does not knowingly collect personal information from children under 13 years of age.

---

## 10. Changes to This Policy

If PlanYT’s functionality changes in the future in a way that affects data usage, this Privacy Policy will be updated accordingly.
